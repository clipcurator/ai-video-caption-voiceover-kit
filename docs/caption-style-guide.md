# Caption Style Guide

Captions should help viewers understand the video without covering the visual focus.

## General rules

- Keep lines short.
- Avoid more than two caption lines at once.
- Use emphasis only for important words.
- Keep punctuation natural for spoken language.
- Do not cover faces, product UI, or important visual details.

## Short-form caption rules

| Rule | Recommendation |
|---|---|
| Line length | 8-14 words for English, 10-18 Chinese characters when possible |
| Duration | Keep each caption readable for the scene pace |
| Emphasis | Highlight names, numbers, contrast words, or plot turns |
| Placement | Lower third by default, adjusted for platform UI |
| Style | Consistent font, outline, and color system |

## Common problems

- Captions are too long for fast scenes.
- Captions repeat the voiceover without visual hierarchy.
- Keyword highlighting is overused.
- Bilingual captions become too dense.
- Captions block faces or action.
