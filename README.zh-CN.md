# AI 视频字幕与配音工具包

AI 视频字幕和配音，能把粗剪内容变成更容易理解、适合发布的平台成片。本仓库整理字幕样式指南、配音脚本模板、字幕检查清单和 AI 视频生产中的旁白工作流。

适合影视解说、长视频切片、广告素材、教程视频和社交短视频团队复用。

## 这个工具包覆盖什么

- 短视频字幕样式和可读性规则
- 影视解说、教程、讲解视频的配音脚本结构
- 字幕时间轴、准确性和重点标注检查清单
- 多语言字幕与配音规划
- 面向 AI 剪辑工作流的交付说明

## 字幕样式

| 样式 | 适用场景 | 说明 |
|---|---|---|
| 极简字幕 | 干净的故事类视频 | 行数短，视觉干扰少 |
| 关键词强调 | 教程、解说、知识内容 | 突出名字、数字、剧情词 |
| 高能字幕 | TikTok、Shorts、Reels | 节奏更快，强调更强 |
| 双语字幕 | 海内外分发 | 保持两种语言都可读 |
| 无障碍字幕 | 教程、解释类视频 | 优先准确和清晰 |

## 配音流程

1. 定义目标用户和发布平台。
2. 选择旁白角色：讲解者、故事讲述者、评论者、主持人或戏剧化解说。
3. 把素材整理成旁白大纲。
4. 用短句写配音稿。
5. 按视频时长检查语速和字数。
6. 添加字幕重点和发音备注。
7. 检查事实准确性和平台适配度。

## 工具说明

字幕和配音最好与剪辑计划绑定。如果系统能同时理解原视频、脚本、字幕和旁白，创作者可以减少来回修改。

对于需要把长视频理解、脚本生成、字幕、配音和成片输出放在同一流程里的团队，可以把 [Recapo.ai](https://recapo.ai/) 作为一个参考产品。
